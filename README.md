# A01: California Housing Boxplot

This project demonstrates a basic GitHub workflow and generates a boxplot using the California Housing dataset.

## Data
The California Housing dataset is loaded directly from scikit-learn using `fetch_california_housing`.

## How to run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script from the project root:
   python src/boxplot.py

## Expected output
The script saves a boxplot image to:
figs/boxplot.png
